Git e GitHub

Git é um controle de versão, usado para criar historico de mudança do codigo

GitHub é uma ferramenta para hospedar código

São duas ferramentas interligadas


Configuração do Git Bash, para conectar o git com github


  1. git confg --global user.name ' seu nome '

  2. git confg --global user.email ' seu email '

  3. cd entrar na pasta do projeto

  4. git clone https...colocar a url do projeto

  5. git add .

  6. git commit -m 'mensagem simples'

  7. git push -u origin main

git add . = Vai adicior as mudanças

git commit -m = Serve para mandar a mudança com uma mensagem simples descrevendo as alterações

git push -u origin main = Código que da um empurrão final para subir pro github

Criar um repositorio no GitHub

Para criar um novo repositório vamos até o perfil do github, clicar em Repositories e no botão verde de New

Preencher os espaços e apertar no botão verde Create repository >
